# Changelog

All notable changes to the Weatherly project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [1.0.0] - 2024-01-XX

### Added
- Initial release of Weatherly weather application
- Current weather display with detailed metrics
- Hourly forecast (48 hours)
- Daily forecast (7-10 days)
- Interactive Google Maps integration
- Location search functionality
- Favorites management
- Auto-location detection
- Multi-language support (English, Spanish)
- Light/Dark/System theme modes
- Unit preferences (Metric/Imperial/Kelvin)
- Offline caching with Hive
- Clean architecture implementation
- Riverpod state management
- Freezed immutable models
- Comprehensive error handling
- Material 3 design system
- Accessibility support

### Technical
- OpenWeatherMap API integration
- Google Maps Platform integration
- Flutter 3.x support
- Null-safety enabled
- Type-safe API clients with Retrofit
- Code generation with build_runner
- Localization with flutter_localizations
